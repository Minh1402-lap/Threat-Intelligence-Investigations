# Threat Intelligence Investigations

This repository contains cybersecurity investigation projects focused on threat intelligence, OSINT, malware analysis, and risk assessment.

## Projects

### Project 01 – Zenfy Investigation

Investigation of a suspicious online recruitment website using:

- VirusTotal
- Hybrid Analysis
- DNS Checker
- ScamAdviser
- Trustpilot

#### Objectives

- Assess malware risk
- Assess phishing risk
- Assess fraud and social engineering risks

#### Findings

- No malware activity identified
- No command-and-control communication observed
- Public reviews raised concerns regarding subscription and billing practices
- Potential fraud and social engineering risks remain

#### Tools Used

- VirusTotal
- Hybrid Analysis
- DNSChecker
